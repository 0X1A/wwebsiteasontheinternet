Getting These Packages

To add these packages to your Pacman repositories you can add this server to your pacman.conf 

	[0X1A]
	SigLevel = Never # See details below about getting signed packages
	Server = http://0x1a.us/pkgs
For signed packages you can add my key to your Pacman keyring using my GPG keyid 4B2076CA. (For more details visit the Arch Wiki). With this change you can set the SigLevel section of the repo to either TrustedOnly or which ever other option you'd like. (More information on SigLevel is available on the Arch Wiki)
